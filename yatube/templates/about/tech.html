Python 3.9.6

Освоен основной синтаксис Python, но требуется углубление знаний в части подключаемых библиотек, полного понимания работы классов, аннотации типов. Да и практики нужно гораздо больше, чтобы почувствовать уверенность.

Django 2.2.6

В данном фреймворке освоены основные элементы, использованные при создании данного портала: внесение изменений в settings, корректное указание путей в urls, создание функций для рендера страниц во views, подключение templates, а также bootstrap и стилей CSS в static, работа с базой данных sqlite3, настройка admin.

Bootstrap 4 and CSS

Пока что я умею только подключать bootstrap со всеми основными компонентами и использовать элементы готовых шаблонов страниц.

HTML5

Освоена элементарная верстка с выводом в шаблон данных из view-функций и view-классов Django.

SQL

Только начал изучать SQL (PostgreSQL, SQLite). Пока что не хватает времени глубоко погрузиться в работу с базами данных, чтобы написать о каких-то конкретных достижениях. Тем не менее, параллельно с основной программой обучения я планирую серьёзно разобраться с PostgreSQL, а в перспективе - пройти отдельные курсы.

GIT

При разработке учебных проектов стараюсь постоянно пользоваться возможностями GIT, чтобы привыкнуть к работе с версиями. Часто использую commit, чтобы была возможность откатить изменения. Новые приложения в проекте разрабатываю в отдельных ветках, а после тестирования работоспособности вливаю в main/master (merge, rebase).
